# FlyHi
Simple flight application for a daily commuter. 
FlyHi is the one stop application for frequent flyers, offering great features such as:
- Flight Access
- Delayed Flight Tracking
- Flight and Car rental packages
- Miles rewards
- Hotel Suggestions
- Tech Support
